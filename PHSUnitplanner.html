About this planner This planner creates one collapsible learning focus plan for each selected curriculum statement. Defaults are generated once, then teachers customise only where needed. LI and SC can use teacher wording or student-friendly wording. The Optional activity ideas can copy the current plan context into ChatGPT or Copilot for activity suggestions.
